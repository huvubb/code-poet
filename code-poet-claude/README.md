# 代码诗人 — Claude Code 版

> 为代码注入诗意 ✨ — Claude Code 直接可用

## 安装（3 秒）

将 `CLAUDE.md` 复制到项目根目录：

```bash
# 当前项目
cp CLAUDE.md /你的项目/

# 全局生效（所有项目）
cp CLAUDE.md ~/.claude/CLAUDE.md
```

Claude Code 启动时自动读取，无需配置。

## 使用

在对话中输入触发词即可：

| 需求 | 怎么说 |
|------|--------|
| 注释美化 | `写诗注释` / `poetic comment` |
| 提交信息 | `诗提交` / `poetic commit` |
| 分隔标题 | `诗标题` / `poetic header` |
| 切换风格 | `换成魏晋风` / `来点幽默的` |

## 文件说明

| 文件 | 用途 |
|------|------|
| `CLAUDE.md` | Claude Code 指令（复制即可用） |
| `README.md` | 本文件 |
