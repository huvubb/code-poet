# 代码诗人 · Code Poet

> 为代码注入诗意 ✨ — 全平台 免费开源

[![Version](https://img.shields.io/badge/version-1.0.0-purple)](https://github.com/huvubb/code-poet/releases)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)
一款将中国古典诗词之美融入程序开发的创意工具，支持五大平台。

## ✨ 功能

| 能力 | 触发词 | 效果 |
|------|--------|------|
| 🖋 诗韵注释 | `写诗注释` / `poetic comment` | 古风盎然的代码注释 |
| 📝 雅致提交 | `诗提交` / `poetic commit` | 文采斐然的 Git commit |
| 📚 篇章标题 | `诗标题` / `poetic header` | 诗词分隔标题 |

四种风格：唐宋风 | 魏晋风 | 现代诗 | 打油诗

## 📦 平台支持

| 平台 | 安装方式 |
|------|----------|
| **CLI** | `npm install -g .` |
| **VS Code** | `code --install-extension code-poet-1.0.0.vsix` |
| **Claude Code** | 复制 `CLAUDE.md` 到项目 |
| **Codex** | 插件市场安装 |
| **GitHub Action** | `uses: huvubb/code-poet/action@main` |

## 🐛 源码

直接克隆即可使用，各平台安装方式见下方。

```
code-poet/
├── code-poet/          Codex 插件
├── code-poet-claude/   Claude Code
├── code-poet-vscode/   VS Code 扩展
├── code-poet-cli/      npm CLI 工具
└── code-poet-action/   GitHub Action
```
