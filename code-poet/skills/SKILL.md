# 代码诗人 (Code Poet)

为代码注入诗意——本技能能将中国古典诗词之美融入日常开发工作流。

## 使用方式

在对话中直接使用以下关键词触发：

### 1. 诗韵注释
为函数、方法和类生成富有文采的注释。

**触发词：** `写诗注释`、`诗意注释`、`poetic comment`

**示例：**
```
用户：用古诗风格给这段代码写注释
代码诗人：已为函数 generateThumbnail 生成注释：
/**
 * 裁云镂月，缩万象于方寸
 * @param {string} srcPath - 原图玉影之所在
 * @param {number} width - 新图之宽度（像素）
 * @returns {Promise<string>} 缩略之图径
 */
```

### 2. 雅致提交
生成文采斐然的 Git 提交信息。

**触发词：** `诗提交`、`文雅提交`、`poetic commit`

**示例：**
```
用户：为我的改动生成一个文雅的 commit message
改动：修复了登录页面在移动端的布局问题
代码诗人：fix(auth): 修竹移影——适配移动端登录布局

登录之页，移于方寸之屏
修竹之姿，应于掌中之器
```

### 3. 篇章标题
用诗词为代码区块创建优雅的分隔标题。

**触发词：** `诗标题`、`诗意分隔`、`poetic header`

**示例：**
```
用户：帮我给这段配置代码加个标题分隔
// ═══════════════════════════════════════
// ｜  天 工 开 物  ·  配 置 之 章       ｜
// ═══════════════════════════════════════
// Configuration Module
// "问渠那得清如许，为有源头活水来"
```

## 创作风格

本技能支持以下风格（默认为**唐宋风**）：

- `唐宋风`：引用唐诗宋词名句 + 仿写
- `魏晋风`：清雅飘逸，多用山水意象
- `现代诗`：白话自由诗风格
- `打油诗`：幽默诙谐，轻松欢快

```
用户：用魏晋风给这个 API 路由加注释
代码诗人：
/**
 * 泛彼柏舟，在河之洲
 * 数据之舟，行于网络之流
 * GET /api/users —— 取众生之相于玉盘
 */
```

## 配置

可通过对话中的关键词切换风格：
- `风格切换 唐宋风`
- `style modern`
- `来点幽默的`
