# 代码诗人 CLI

> 命令行代码注释美化工具 — ¥10

## 安装

```bash
npm install -g code-poet
```

## 使用

```bash
code-poet comment                    # 生成诗韵注释
code-poet commit                     # 生成雅致提交信息
code-poet header --style wei-jin     # 生成篇章标题（魏晋风）

诗 comment                           # 中文命令也支持
```

## 风格

`--style tang-song | wei-jin | modern | humor`
